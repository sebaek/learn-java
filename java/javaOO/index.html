Lesson: 클래스와 객체

자바 언어의 기초를 배웠다. 이제 클래스를 작성하는 법을 배울 차례이다. 이 강좌에서는 클래스를 정의하는 법과 멤버 변수, 메소드, 생성자를 선언하는 방법을 배울 것이다.

그리고 객체를 생성하기 위해 클래스를 사용하는 방법과 생성된 객체를 사용하는 방법을 배운다.

중첩 클래스와 열거형에 대해서도 배운다.

클래스

이 섹션에서는 클래스를 파헤쳐보고 필드와 메소드, 생성자를 선언하는 법을 배운다.

객체

이 섹션은 객체를 생성하고 사용하는 방법에 대해 배운다. 객체를 초기화하는 법과 인스턴스 변수와 메소드를 접근하는 dot연산자를 사용하는 방법을 배운다.

More on Classes

이 섹션은 메소드의 값 반환하기, this키워드, 클래스 멤버와 인스턴스 멤버의 차이, 접근 제어에 대해 배운다.

중첩 클래스

static 중첩 클래스, 내부 클래스, 무명 내부 클래스, 지역 클래스, 람다 표현식에 대해 배운다. 또하 이 것을 언제 사용해야하는 지 알려준다.

Enum Types

이 섹션은 상수의 집합을 정의하고 사용할 수 있는 특별한 클래스인 열거형에 대해 배운다.
