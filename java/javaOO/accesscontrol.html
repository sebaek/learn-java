클래스의 멤버 접근 제어하기

접근 지정자는 특정 필드를 사용하거나 특정 메소드를 사용할 수 있는 다른 클래스를 결정한다. 두 가지 레벨의 접근 제어가 있다.

  • At the top level - public 또는 package-private(지정자 없음)
  • At the member level - public, private, protected 또는 package-private(지정자 없음)

Access Levels
Modifier Class Package Subclass World
public Y Y Y Y
protected Y Y Y N
no modifier Y Y N N
private Y N N N
