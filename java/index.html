객체 지향 프로그래밍 개념

이 섹션은 객체 지향 프로그래밍의 핵심 개념에 대해 알려준다. 객체, 메세지, 클래스와 상속 등. 이 강의는 어떻게 이런 개념들이 코드로 작성되는지 보여준다. 이미 객체 지향 프로그래밍에 익숙하다면 건너 뛰어도 된다.

언어의 기본

프로그래밍 언어의 전통적인 기능에 대해 설명한다. 변수, 배열, 자료형, 연산자, 실행 흐름 등.

클래스와 객체

객체를 만들어 내는 클래스를 어떻게 작성하는 지 알아보고, 객체를 생성하고 사용하는 방법을 배워본다.

어노테이션

어노테이션은 컴파일러에게 제공하는 정보이며 메타데이터 형식이다. 이 강의는 프로그램에서 어노테이션을 어떻게 작성하고 사용하는 지 알려준다.

인터페이스와 상속

인터페이스가 무엇이며 왜 사용하는지 어떻게 작성하는 지 배워본다. 이 섹션은 어떤 클래스가 다른 클래스로부터 파생되는 것을 보여준다. 즉 어떻게 서브클래스가 슈퍼클래스로부터 필드와 메소드를 상속 받을 수 있는 지 알려준다. Object클래스로 부터 파생된 모든 클래스에 대해 배울 것이고 슈퍼클래스를 상속한 서브클래스의 메소드를 어떻게 수정하는 지 배워본다.

수와 문자열

이 강의에서 Number와 String객체를 사용하는 방법을 배워본다. 또한 데이터 출력을 위해 포맷팅 하는 방법을 보여준다.

제네릭

자바 언어에서의 강력한 기능이다. 작성하는 코드에서 자료형을 안전하게 사용하고 컴파일 시에 버그를 찾을 수 있도록 해준다.

패키지

자바 언어에서 클래스들을 분류하고 관계있는 것끼리 묶을 수 있도록 하는 기능이다.
