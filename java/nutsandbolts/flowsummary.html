흐름 제어문 요약

if-then문은 가장 기초적인 제어문이다. 이 문은 어떤 조건이 true일 때만 특정 코드를 실행하게 한다. if-then-else문은 "if"구문의 조건의 false일 때 실행할 다른 코드 섹션을 제공한다. if-then과 if-then-else와 다르게 switch문은 여러 실행흐름을 제공할 수 있다. while문과 do-while문은 조건식이 true일 경우 명령문 블럭이 계속 실행된다. while과 do-while문의 차이는 전자는 조건식이 위에 있고 후자는 조건식이 아래에 있는 것이다. 그래서 do-while의 코드 블럭은 적어도 한 번 실행된다. for문은 일정한 범위의 값을 탐색할때 효과적이다. 두 가지 형식이 있으며, 하나는 콜렉션과 배열을 탐색하는데 유용하다.
