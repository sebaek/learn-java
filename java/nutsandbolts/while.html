while과 do-while문

while문은 특정 조건이 true인 동안 명령문의 블럭이 게속 실행된다. 문법은 다음과 같다. 

while (expression) {
    statement(s)
}

while은 boolean 값을 리턴하는 식을 사용한다. 계산식이 true이면 while문은 명령문 블럭을 실행한다. 계산식이 false가 될 때까지 계산식과 명령문 블럭을 계속 실행한다. 아래 WhileDemo 프로그램은 while 문을 사용해서 1부터 10까지 출력하는 예제이다. 

class WhileDemo {
    public static void main(String[] args){
        int count = 1;
        while (count < 11) {
            System.out.println("Count is: " + count);
            count++;
        }
    }
}

아래 while 문은 무한루프를 구현한 것이다. 

while (true){
    // your code goes here
}

자바 언어는 아래 코드처럼 작성할 수 있는 do-while문도 있다. 

do {
    statement(s)
} while (expression);

while문과의 차이점은 계산식이 반복문의 아래 쪽에 있다는 것이다. 그래서 do블럭은 적어도 한 번 실행된다. 아래 DoWhileDemo 프로그램에서 보여준다. 

class DoWhileDemo {
    public static void main(String[] args){
        int count = 1;
        do {
            System.out.println("Count is: " + count);
            count++;
        } while (count < 11);
    }
}
