변수 요약

자바 언어에서는 "필드"와 "변수"를 모두 사용한다. 인스턴스 변수(non-static 필드)는 클래스의 각 인스턴스마다 갖는 변수이다. 클래스 변수 (static 필드)는 static지정자와 함께 선언되며, 하나의 클래스에 하나의 변수만 존재한다. 지역변수는 메소드 안의 임시 상태를 저장한다. 매개변수는 메소드에게 제공되는 부가 정보이다. 지역변수와 매개변수는 항상 ("필드"가 아닌)"변수"로 분류된다. 필드와 변수를 이름 지을 때 따라야 하는 규칙과 관습이 있다.

8개의 원시 자료형은 : byte, short, int, long, float, double, boolean, char 가 있다. java.lang.String클래스는 문자열을 나타낸다. 컴파일러는 위의 타입에 기본값을 필드에 할당 할 수 있다. 지역변수에는 기본 값을 할당하지 않는다. 리터럴은 소스코드에 표현된 고정된 값이다. 배열은 같은 자료형의 고정된 갯수의 여러 원소를 담을 수 있는 객체이다. 배열의 길이는 배열이 생성될 때 정해지며 변하지 않는다.
