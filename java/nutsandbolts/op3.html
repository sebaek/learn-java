비트, 쉬프트 연산자

자바 언어에서도 정수형 타입의 비트 연산자와 쉬프트 연산자를 제공한다. 이 섹션에서 다루는 연산자는 자주 사용되지 않는다. 그래서 간략하게 다룰 것이며, 이런 종류의 연산자가 존재한다는 것을 알려주는 섹션이다.

단항 비트 연산자 "~"는 비트의 패턴을 반전 시킨다. 모든 정수형 타입의 0비트를 1로 1비트를 0으로 바꾼다. 예를 들어 8비트의 byte 값을 보면, "00000000"에 이 연산자를 적용하면 "11111111"이 된다.

signed left shift 연산자 "<<"는 비트의 패턴을 왼쪽으로 이동시키고 signed left shift 연산자 ">>"는 비트의 패턴을 오른쪽으로 이동시킨다. 이동할 비트 패턴은 왼쪽 피연산자로 두고, 이동할 포지션은 오른쪽 피연산자로 둔다. unsigned right shift 연산자 ">>>"는 왼쪽 끝을 0으로 채우지만 ">>"는 부호 비트로 채운다.

& 연산자는 bitwise AND 연산자이다.

^ 연산자는 bitwise exclusive OR 연산자이다.

| 연산자는 bitwise inclusive OR 연산자이다.

BitDemo 프로그램은 bitwise AND 연산하여 수 "2"를 출력하는 예제이다. 

class BitDemo {
    public static void main(String[] args) {
        int bitmask = 0x000F;
        int val = 0x2222;
        // prints "2"
        System.out.println(val & bitmask);
    }
}
