할당, 산술, 단항 연산자

간단한 할당 연산자

가장 많이 마주하게 되는 연산자 중에 하나는 간단한 할당연산자 "="일 것이다. 이 연산자를 Bicycle 클래스에서 보았다. 오른쪽에 있는 값을 왼쪽에 있는 피연산자에 할당한다. 

int cadence = 0;
int speed = 0;
int gear = 1;

이 연산자는 객체의 참조를 할당 할 때도 사용한다. 다른 섹션에서 배우도록 하겠다.

산술 연산자

자바는 덧셈, 뺄셈, 곱셈, 나눗셈을 위한 연산자를 제공한다. 기초수학에서 배운 연산자와 같고, " %"는 처음 볼 것이다. 이 연산자는 어떤 값을 다른 값으로 나눈 나머지를 결과로 갖는다.

Operator Description
+ 덧셈 (문자열에선 연결)
- 뺄셈
* 곱셈
/ 나눗셈
% 나머지

아래 ArithmeticDemo프로그램에서 산술연산자를 사용한다.

class ArithmeticDemo {

    public static void main (String[] args) {

        int result = 1 + 2;
        // result is now 3
        System.out.println("1 + 2 = " + result);
        int original_result = result;

        result = result - 1;
        // result is now 2
        System.out.println(original_result + " - 1 = " + result);
        original_result = result;

        result = result * 2;
        // result is now 4
        System.out.println(original_result + " * 2 = " + result);
        original_result = result;

        result = result / 2;
        // result is now 2
        System.out.println(original_result + " / 2 = " + result);
        original_result = result;

        result = result + 8;
        // result is now 10
        System.out.println(original_result + " + 8 = " + result);
        original_result = result;

        result = result % 7;
        // result is now 3
        System.out.println(original_result + " % 7 = " + result);
    }
}

결과는 : 

1 + 2 = 3
3 - 1 = 2
2 * 2 = 4
4 / 2 = 2
2 + 8 = 10
10 % 7 = 3

산술연산자를 간단한 할당연산자와 결합하여 사용할 수 있다. 예를 들어 x+=1;과 x=x+1;은 x를 1 증가 시킨다.

+연산자는 두 문자열을 연결 시키는 기능도 있다. 아래 프로그램 ConcatDemo에서 보여준다. 

class ConcatDemo {
    public static void main(String[] args){
        String firstString = "This is";
        String secondString = " a concatenated string.";
        String thirdString = firstString+secondString;
        System.out.println(thirdString);
    }
}

프로그램 마지막 부분에 변수 thirdString은 "This is a concatenated string."을 가지고 있고, 그 값이 출력된다.

단항 연산자

단항 연산자는 피연산자가 하나만 존재한다. 1씩 증가 시키거나 감소시키는 연산자, 결과의 부호를 바꾸는 연산자, 불린값을 반전시키는 연산자 등이 있다.

Operator Description
+ Unary plus operator; indicates positive value (numbers are positive without this, however)
- Unary minus operator; negates an expression
++ Increment operator; increments a value by 1
-- Decrement operator; decrements a value by 1
! Logical complement operator; inverts the value of a boolean

아래 UnaryDemo 프로그램에서 단항 연산자를 보여준다. 

class UnaryDemo {

    public static void main(String[] args) {

        int result = +1;
        // result is now 1
        System.out.println(result);

        result--;
        // result is now 0
        System.out.println(result);

        result++;
        // result is now 1
        System.out.println(result);

        result = -result;
        // result is now -1
        System.out.println(result);

        boolean success = false;
        // false
        System.out.println(success);
        // true
        System.out.println(!success);
    }
}

증감 연산자는 피연산자의 앞에나 뒤에 붙을 수 있다. result++;와 ++result;는 모두 result를 1 증가 시킨다. 차이점은 ++result는 증가된 값으로 계산하고, result++는 원래 값으로 계산한다는 것만 있다. 단순히 증감만 하기 위해 사용한다면 어느 것을 사용해도 무방하다. 하지만 이 연산자를 좀 더 복잡한 식에 사용하면 조금 다른 결과가 나온다.

아래 PrePostDemo프로그램은 prefix/postfix 단항 연산자를 사용하는 것을 보여준다. 

class PrePostDemo {
    public static void main(String[] args){
        int i = 3;
        i++;
        // prints 4
        System.out.println(i);
        ++i;			   
        // prints 5
        System.out.println(i);
        // prints 6
        System.out.println(++i);
        // prints 6
        System.out.println(i++);
        // prints 7
        System.out.println(i);
    }
}
