switch문

if-then과 if-then-else문과 다르게, switch문은 여러개의 실행 경로를 가질 수 있다. switch문은 byte, short, char와 int 원시 자료형과 같이 사용한다. 또한 열거형(enumerated type)과 String클래스, 원시 자료형을 감싸는(wrap) 클래스( Character, Byte, Short, Integer)들을 사용할 수 있다.

아래 예제 SwitchDemo는 달의 값을 갖는 month라는 int타입 변수를 선언했다. 프로그램은 switch문에서 month의 값에 따라 달을 출력하는 프로그램이다. 

public class SwitchDemo {
    public static void main(String[] args) {

        int month = 8;
        String monthString;
        switch (month) {
            case 1:  monthString = "January";
                        break;
            case 2:  monthString = "February";
                        break;
            case 3:  monthString = "March";
                        break;
            case 4:  monthString = "April";
                        break;
            case 5:  monthString = "May";
                        break;
            case 6:  monthString = "June";
                        break;
            case 7:  monthString = "July";
                        break;
            case 8:  monthString = "August";
                        break;
            case 9:  monthString = "September";
                        break;
            case 10: monthString = "October";
                        break;
            case 11: monthString = "November";
                        break;
            case 12: monthString = "December";
                        break;
            default: monthString = "Invalid month";
                        break;
        }
        System.out.println(monthString);
    }
}

결과는 August가 출력된다.

switch의 몸통은 switch 블럭으로 불린다. switch 블럭의 명령문은 하나이상의 case나 default로 라벨을 붙일 수 있다. switch문은 식을 계산하고 결과를 case 라벨의 값과 일치하는 곳의 명령문을 실행한다.

달을 출력하는 예제를 if-then-else문을 사용해서 만들 수 있다. 

int month = 8;
if (month == 1) {
    System.out.println("January");
} else if (month == 2) {
    System.out.println("February");
}
...  // and so on

if-then-else이나 switch중 어떤 것을 사용할 지는 가독성과 조건을 계산하는 식에 따라 결정된다. if-then-else는 값의 조건이나 범위를 판단할 수 있고, switch는 정수 값이나 열거형의 값 또는 String 값 자체을 판단한다.

다른 흥미로운 점은 break문이다. 각 break문은 switch문을 종료 시킨다. switch 블럭은 첫 명령문부터 계속 실행된다. break문이 없으면 명령문은 그 다음에 오는 case의 명령문까지도 계속 실행되기 때문에 break문이 필요하다. SwitchDemoFallThrough 프로그램은 switch블럭이 계속 실행되는 예제를 보여준다. 프로그램은 month의 값과 일치하는 달과 그 이후의 달까지 모두 출력한다. 

public class SwitchDemoFallThrough {

    public static void main(String[] args) {
        java.util.ArrayList futureMonths =
            new java.util.ArrayList();

        int month = 8;

        switch (month) {
            case 1:  futureMonths.add("January");
            case 2:  futureMonths.add("February");
            case 3:  futureMonths.add("March");
            case 4:  futureMonths.add("April");
            case 5:  futureMonths.add("May");
            case 6:  futureMonths.add("June");
            case 7:  futureMonths.add("July");
            case 8:  futureMonths.add("August");
            case 9:  futureMonths.add("September");
            case 10: futureMonths.add("October");
            case 11: futureMonths.add("November");
            case 12: futureMonths.add("December");
                        break;
            default: break;
        }

        if (futureMonths.isEmpty()) {
            System.out.println("Invalid month number");
        } else {
            for (String monthName : futureMonths) {
                System.out.println(monthName);
            }
        }
    }
}

출력 결과는 다음과 같다. 

August
September
October
November
December

엄밀히 말하면 switch문이 바로 종료되기 때문에 마지막 break문은 필요하지 않다. 코드를 쉽게 수정하고 에러의 원인을 줄이기 위해서 break문을 사용하는 것은 권장된다. default섹션은 case의 값과 일치 하지 않는 모든 값에 해당한다.

SwitchDemo2프로그램은 여러개의 case 라벨을 사용하는 방법을 보여준다. 특정한 달의 몇 개의 날이 있는 지 계산하는 프로그램이다. 

class SwitchDemo2 {
    public static void main(String[] args) {

        int month = 2;
        int year = 2000;
        int numDays = 0;

        switch (month) {
            case 1: case 3: case 5:
            case 7: case 8: case 10:
            case 12:
                numDays = 31;
                break;
            case 4: case 6:
            case 9: case 11:
                numDays = 30;
                break;
            case 2:
                if (((year % 4 == 0) && 
                        !(year % 100 == 0))
                        || (year % 400 == 0))
                    numDays = 29;
                else
                    numDays = 28;
                break;
            default:
                System.out.println("Invalid month.");
                break;
        }
        System.out.println("Number of Days = "
                            + numDays);
    }
}

결과는 

Number of Days = 29

switch문에서 String 사용하기

자바 SE 7 이후에 switch문에서 String을 사용할 수 있게 되었다. 아래 StringSwitchDemo 프로그램에서 month라는 String타입의 변수의 값의 따라 달의 숫자를 출력하는 예제를 보여준다. 

public class StringSwitchDemo {

    public static int getMonthNumber(String month) {

        int monthNumber = 0;

        if (month == null) {
            return monthNumber;
        }

        switch (month.toLowerCase()) {
            case "january":
                monthNumber = 1;
                break;
            case "february":
                monthNumber = 2;
                break;
            case "march":
                monthNumber = 3;
                break;
            case "april":
                monthNumber = 4;
                break;
            case "may":
                monthNumber = 5;
                break;
            case "june":
                monthNumber = 6;
                break;
            case "july":
                monthNumber = 7;
                break;
            case "august":
                monthNumber = 8;
                break;
            case "september":
                monthNumber = 9;
                break;
            case "october":
                monthNumber = 10;
                break;
            case "november":
                monthNumber = 11;
                break;
            case "december":
                monthNumber = 12;
                break;
            default: 
                monthNumber = 0;
                break;
        }

        return monthNumber;
    }

    public static void main(String[] args) {

        String month = "August";

        int returnedMonthNumber =
            StringSwitchDemo.getMonthNumber(month);

        if (returnedMonthNumber == 0) {
            System.out.println("Invalid month");
        } else {
            System.out.println(returnedMonthNumber);
        }
    }
}

결과는 8이다.

switch에서 String은 각 case 라벨의 값을 String.equals 메소드를 사용해서 비교한다. StringSwitchDemo에서는 대소문자 구분없이 검사하기 위해 month의 값을 toLowerCase메소드를 사용해서 소문자로 바꾸고, case 라벨의 값을 모두 소문자로 작성하였다.

Note : 이 예제에서는 switch문의 식의 null인지 검사 하였다. NullPointerException이 발생하지 않도록 모든 switch문의 검사식이 null이 되지 않게 주의해야 한다.
