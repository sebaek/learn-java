자바로 무엇을 할 수 있나?

일반적인 목적으로 자바 프로그래밍 언어는 강력한 소프트웨어 플랫폼이다. 자바 플랫폼은 아래 기능들을 제공한다.

  • 개발 도구 : 개발 도구는 컴파일, 실행, 모니터링, 디버깅, 문서화등 프로그램 개발에 필요한 모든 도구를 제공한다. 처음 사용하는 주요 툴은 javac컴파일러, java 실행기, javadoc 문서화 도구 일 것이다.
  • Application Programming Interface (API) : API는 자바 프로그래밍 언어의 핵심 기술을 제공한다. 직접 개발하는 프로그램에 바로 사용 가능한 유용한 클래스들의 모음이다. 기초적인 객체부터 네트워크 보안, 데이터베이스 접속, XML 출력등 모든 것을 묶고 있다. 핵심 API는 매우 방대하다. Java Platform Standard Edition 8 Documentation에 접속하면 모든 것을 볼 수 있다.
  • 배포 기술 : JDK 소프트웨어는 Java Web Start 소프트웨어와 Java Plug-In 소프트웨어 같은 프로그램을 엔드유저에게 배포하기위한 표준 메카니즘을 제공한다.
  • User Interface 도구 : The JavaFX, Swing, Java 2D toolkis 은 그래픽 유저 인터페이스(GUI)를 만들 수 있는 도구 이다.
  • 통합 라이브러리 : Integration libraries such as the Java IDL API, JDBC API, Java Naming and Directory Interface (JNDI) API, Java RMI, and Java Remote Method Invocation over Internet Inter-ORB Protocol Technology (Java RMI-IIOP Technology) enable database access and manipulation of remote objects.
