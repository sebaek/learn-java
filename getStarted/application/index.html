"Hello World!" 프로그램 살펴보기

"Hello World!"프로그램을 작성하고 컴파일 해보았다. 이 것이 어떻게 작동하는지 알아보자. 

class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

"Hello World!" 프로그램은 세 개의 주요 구성요소로 이루어져 있다 : 소스 코드 주석, HelloWorldApp 클래스 정의와 main메소드. 아래 설명은 코드를 이해하는 데 기본적인 내용이 될 것이다. 더 많은 내용은 나머지 튜토리얼을 읽은 후에 알게 될 것이다.

소스코드 주석

아래 굵은 글씨는 주석을 의미 한다. 

/**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

주석은 컴파일러에 의해 무시되지만 프로그래머들에게 유용하다. 자바 프로그래밍 언어는 세 종류의 주석을 제공한다.

/* text */
컴파일러는 /* 과 */ 사이의 모든 것을 무시한다.
/** documentation */
이 것은 문서화 주석을 의미한다. 컴파일러는 /** 과 */ 사이의 모든 것을 무시한다. javadoc 도구는 문서화 주석을 사용하여 문서를 생성한다. 더 많은 정보는 Javadoc™ tool documentaion을 참조.
// text
컴파일러는 //부터 줄의 끝까지 무시한다.

HelloWorldApp 클래스 정의

아래 굵은 텍스트는 "Hello World!"프로그램에 사용된 클래스 정의 블럭이다. 

/**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

위에서 보여주듯이 클래스를 정의 하는 기본 형식은 아래와 같다. 

class name {
    . . .
}

class키워드는 이름이 name인 클래스를 정의 하기위한 시작 부분이다. 그리고 클래스의 코드는 중괄호 사이에 작성된다. 다른 튜토리얼에서 클래스의 일반적인 내용과 상세한 내용을 알아보도록 하겠다. 지금은 모든 프로그램은 클래스 정의와 함께 시작된다는 것을 안 것으로 충분하다.

main 메소드

아래 굵은 글씨는 main 메소드를 정의 하는 부분이다. 

/**
 * The HelloWorldApp class implements an application that
 * simply displays "Hello World!" to the standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); //Display the string.
    }
}

자바 언어에서 모든 프로그램은 다음 시그니쳐를 가진 main 메소드를 가지고 있어야 한다. 

public static void main(String[] args)

public과 static 지정자는 순서를 바꿔도 된다(public static 또는 static public). 그러나 public static으로 쓰는 것이 일반적인 방법이다. 매개변수의 이름은 원하는 대로 지을 수 있다. 하지만 많은 프로그래머들이 "args"나 "argv"를 사용한다.

main메소드는 하나의 매개변수를 갖는다. String 배열 타입이다. 

public static void main(String[] args)

이 배열은 프로그램을 실행할 때 시스템에서 전달하는 정보를 갖는다. 예를 들어: 

java MyApp arg1 arg2

배열에 있는 각 문자열을 command-line argument(명령문 인자)라고 한다. 명령문 인자는 컴파일을 다시 하지 않고 사용자가 프로그램 실행에 영향을 주기 위해 사용한다. 예를 들어 정렬 프로그램에서 사용자는 데이터의 정렬 방식을 명령문 인자를 통해 전달 할 수 있다. 

-descending

"Hello World!" 프로그램은 이 명령문 인자를 무시하지만 이런 인자가 있다는 것을 알고 있어야 한다.

마지막으로 설명할 줄 : 

System.out.println("Hello World!");

이 줄은 핵심 라이브러리의 System 클래스를 사용하여 "Hello World!"를 표준 출력장치에 출력한다. 이 라이브러리에 관한 내용은 ("Application Programming Interface", 또는 "API" 라고 불림) 나중에 자세히 다룰 것이다.
