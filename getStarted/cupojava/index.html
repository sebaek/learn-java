첫 번째 프로그램 만들기

첫 번째 프로그램은 HelloWorldApp이다. 간단히 "Hello World!"라고 출력하는 프로그램이다. 이 프로그램을 만들기 위해서 :

  • 소스 파일을 만든다.

    소스 파일은 자바 프로그래밍 언어로 작성된 코드를 담고 있다. 작성자와 다른 프로그래머가 이해할 수 있는 코드 이다. 소스 파일은 텍스트 에디터로 작성하고 편집할 수 있다.

  • 소스 파일은 .class 파일로 컴파일 한다.

    자바 프로그래밍 언어 컴파일러( javac)는 소스 파일의 텍스트를 번역해서 자바 가상 머신이 이해 할 수 있는 명령문으로 바꿔준다. 이 파일에 담겨 있는 명령문들은 바이트코드라고 한다.

  • 프로그램을 실행 시킨다.

    자바 프로그램 실행도구( java)는 자바 가상 머신이 작성된 프로그램을 실행 시킬 수 있게 한다.

소스 파일 만들기

먼저, 편집기를 실행 시킨다. 메모장을 실행 시키도록 한다. 새 문서에서 아래와 같은 코드를 작성한다. 

/**
 * HelloWorldApp 클래스는
 * 간단히 표준 출력에 "Hello World!"라고 출력하는
 * 프로그램을 구현했다.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // 문자열 출력.
    }
}

주의 : 모든 코드와 명령문, 파일이름을 보이는 대로 정확하게 작성하여야 한다. 컴파일러( javac)와 실행프로그램( java)는 모두 대소문자를 명확히 구분하기 때문에 코드 작성시 주의하여야 한다.

작성된 파일을 HelloWorldApp.java로 저장한다.

소스 파일을 .class 파일로 컴파일 하기

win + r키를 눌러 실행 창을 띄운 뒤, cmd를 입력하고 엔터 키를 누르면 아래와 같은 쉘 윈도우가 나타날 것이다.

쉘 윈도우

프롬프트는 현재 디렉토리를 보여준다. 소스 파일이 있는 위치로 이동한다.

디렉토리를 이동하여 컴파일할 준비가 되었으면 아래 명령어를 작성하고 엔터키를 누른다. 

javac HelloWorldApp.java

컴파일러는 HelloWorldApp.class 바이트코드 파일을 생성했다. 프롬프트에서 dir명령어를 입력하여 파일이 생성된 것을 볼 수 있다. 이 파일을 실행 할 수 있다.

프로그램 실행하기

같은 디렉토리에서 아래 명령어를 실행시킨다. 

java -cp . HelloWorldApp

스크린에 Hello World!라고 출력될 것이다.
